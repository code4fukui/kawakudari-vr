# kawakudari-vr

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A simple VR game where the player avoids obstacles in a downward scrolling environment.

## Demo
You can play the game in your web browser by visiting the project's GitHub page.

## Features
- Immersive VR experience using A-Frame
- Randomly generated obstacles that the player must avoid
- Scrolling background that creates a sense of movement
- Scoring system that tracks the player's progress

## Requirements
This project is designed to run in a web browser and uses the A-Frame library for VR functionality. No additional software or hardware is required, other than a compatible VR headset (if the user wishes to experience the game in VR).

## Usage
1. Clone the repository or download the ZIP file.
2. Open the `index.html` file in a web browser.
3. Use the arrow keys or head movements (if in VR mode) to navigate the player and avoid the obstacles.
4. The game will end when the player collides with an obstacle or reaches the end of the level.
