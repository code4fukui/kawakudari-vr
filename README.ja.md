# かわくだりVR

このプロジェクトは、VR向けの簡単なゲーム「かわくだり」を実装したものです。プレイヤーは障害物を避けながら、下方向に進み続けることが目的です。一定の距離に到達するとゲームがクリアになります。

## デモ
[デモページ](https://fukuno.jig.jp/app/kawakudari-vr/)で動作を確認できます。

## 機能
- VRモードで没入的なゲーム体験が可能
- ランダムに生成される障害物を避ける
- 下方向にスクロールするバックグラウンドで移動感を演出
- プレイヤーの進捗を記録するスコア機能

## 必要環境
このプロジェクトはWebブラウザ上で動作し、A-Frameライブラリを使ってVR機能を実現しています。VRヘッドセットがあれば、よりimmersiveな体験ができます。

## 使い方
1. リポジトリをクローンするか、ZIPファイルをダウンロードする
2. `index.html`ファイルをWebブラウザで開く
3. 矢印キーやヘッド操作（VRモード時）を使って、プレイヤーを操作し障害物を避ける
4. プレイヤーが障害物に当たるか、ゲームエリアの終端に到達するとゲームオーバー
